COVID-19

Covid Clade Membership (Pie Chart) & Death Toll (Line Chart)

Select Month of Clade Membership Worldwide (Range: 1- 20)

Description

To answer question 2 "Analyse the Covid-19 mutations worldwide and their influence on the death toll in the UK. Is there any relationship between them?". To answer this question, design 2 is implemented. There are two charts created for visualisation and analysis. One is pie chart about mutation clade membership of Covid-19, another is the line chart showing the death toll each month in the UK. Through observing the charts, users can have understandings in the relationship between the trend of death toll and the clade membership worldwide.

Data Processing

For data processing, the data for the pie chart is .tsv file, and it is already sorted by date in ascending order before imported into script, and read through d3.tsv(). For the data used for line chart, the .csv file is by d3.csv() as well. The tsv file are stored into dictionary,including date and clade membership. Then, select range of date for each month, and count the number of each clade (e.g. '20A', '20B'). For the csv file, since the original file includes the accumulative death number, that death toll for each month need to be generated manually. The calculation is based on the subtraction between the value of end date and start date of each month. For drawing data on charts, all attributes are aligned with corresponding date. The attribute used for "nextstrain_ncov_open_global_metadata.tsv" is "date" and "clade_membership". The attributes used for "overview_2022-04-08.csv" are "date" and "cumDeaths28DaysByDeathDate".
